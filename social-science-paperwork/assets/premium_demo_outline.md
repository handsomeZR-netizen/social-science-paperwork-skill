# Premium Draft Demo Internal Outline

Use this outline before writing LaTeX.

## One-Sentence Argument

In [setting], this demo shows how [topic] can be studied through [approach], supported by [real inputs + simulated evidence], within [boundary].

## Claim-Evidence Map

| Claim | Evidence type | Status |
|---|---|---|
| [claim] | [literature/statistic/interview/case/survey] | real/proposed/simulated/TODO |

## Manuscript Spine

1. Context and tension.
2. Bounded literature gap.
3. Conceptual framework.
4. Research questions.
5. Proposed design.
6. Illustrative findings.
7. Discussion and implication.
8. Demo-to-real replacement path.

## Required Outputs

- LaTeX paper package.
- Compiled PDF if possible.
- `compile_status.md` with compiler, pass count, PDF status, and important warnings.
- `audit_script_results.md` for questionnaire/codebook/reference audit scripts, or a reason the scripts were not applicable.
- Quality audit.
- Demo-to-real checklist.
- Presentation/advisor report.

## Literature Boundary

Use these labels:

- `REAL-VERIFIED` - full text or source excerpt checked; can support a claim.
- `REAL-METADATA-ONLY` - real metadata only; cannot support a claim yet.
- `VERIFY` - candidate paper; must be checked.
- `TODO-CITATION` - claim needs a source.
- `DRAFT-SIM` - simulated demo material.

Separate verified references from candidate references and citation TODOs in the manuscript or audit file.

## Presentation Order

1. State the Draft Demo boundary.
2. Show the real input.
3. Explain the chosen research route.
4. Show the artifacts.
5. Show the audit and risk controls.
6. Show the minimum viable real-study plan.

## Minimum Viable Real-Study Plan

| Step | Target | Output |
|---|---|---|
| Literature verification | 8-12 core papers | Verified matrix with claim support |
| Case selection | 2-3 cases/artifacts | Case rationale |
| Ethics route | Approval or exemption decision | Consent/data plan |
| Pilot interviews | 2 interviews | Revised protocol |
| Formal interviews | 6-8 interviews | Anonymized excerpts |
| Coding | 1 revision cycle | Codebook and theme memo |
| Replacement | Remove all demo markers | Real draft |
